Transformers 2013-2014

This project is my final year project in university developed using Python on Blender. It explores automated 3D mesh cutting respecting defined aspect ratio and skeletal topology.

Elemental Tower Defense May 2012

This is a project my team built during a school course. It is a tower defense game that runs on iPad built with objective-C. In this game players build advanced towers through combining basic elements and we implemented a social interaction feature allowing Facebook friends to attack each other. This game features common attack types from tower defense games in Warcraft, such as splash aoe, freezing, split attack as well as creep interactions such as enrage, protect.

Woof Vs Meow! May 2013

My team built this project during a school course. It is a multiplayer turn based strategy game built using C# on Unity featuring cartoon cats and dogs at war. This project won us the 1st place in 2nd SOC Term Project Showcase.
